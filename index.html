DevTools

DevTools is a Roblox Studio plugin that I wrote to simplify actions while developing. It has many features and more are currently being developed.

RoEncode

RoEncode is a simple string encryption module that can be used for in-game puzzles and more. This project has been discontinued and will not recieve any future updates.

Build Battles

Build Battles is a competition game where players are given a topic to build on a plot. After a few minutes, players are allowed to fly to other builds and rate their quality. This game is heavily inspired by Hypixel's MC Build Battles!
